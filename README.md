# Accredian Enterprise Clone

A full-stack landing page clone built for the internship assignment.

## 🚀 Features
*   **Framework**: Next.js (App Router)
*   **Styling**: Tailwind CSS
*   **Full Stack**: Built-in API route for lead capture.
*   **Bonus Points**: Integrated a functional form submission backend.

## 🤖 AI Usage Transparency
*   Used an AI assistant to scaffold layout structures quickly.
*   Leveraged AI for rapid debugging of Git path errors.

## 🛠️ Setup Instructions
1. Clone the repo: `git clone YOUR_GITHUB_URL`
2. Install packages: `npm install`
3. Run locally: `npm run dev`
